1. Introduction

1.1 Historical Context and Motivation

The Riemann Hypothesis (RH), formulated by Bernhard Riemann in 1859, remains one of mathematics' most profound unsolved problems and represents one of the seven Clay Institute Millennium Prize Problems [1]. The hypothesis asserts that all non-trivial zeros of the Riemann zeta function Œ∂(s) have real part equal to 1/2, with implications extending far beyond pure number theory into areas including prime number distribution, cryptography, and quantum physics [2].

Computational verification of RH predictions has been a major focus of mathematical research for over five decades. Pioneering work by Lehmer [3] in the 1950s established the foundation for large-scale numerical investigations, while subsequent efforts by Odlyzko [4, 5], Gourdon [6], and others have pushed verification to astronomical scales, confirming RH predictions for the first 1013 non-trivial zeros.

However, these computational achievements have been largely confined to specialized high-performance computing environments, creating significant barriers for educational institutions and independent researchers. The computational requirements‚Äîoften involving months of supercomputer time and specialized mathematical software‚Äîhave limited accessibility to advanced RH exploration.

1.2 Browser-Based Scientific Computing Revolution

Recent advances in JavaScript performance and web browser capabilities have opened new possibilities for scientific computation [7, 8]. Modern JavaScript engines achieve performance within an order of magnitude of compiled languages for numerical tasks, while maintaining universal accessibility across platforms and devices [9].

The democratization of computational mathematics through browser-based implementations addresses a critical gap in mathematical education and research accessibility. Unlike traditional computational approaches requiring specialized hardware and software installations, browser-based solutions offer immediate access to advanced mathematical exploration tools.

A.1 Complete M√∂bius Sieve Implementation

function mobiusSieve(N) { const mu = new Int8Array(N + 1); const lp = new Int32Array(N + 1); mu[1] = 1; const primes = []; for (let x = 2; x <= N; x++) { if (lp[x] === 0) { lp[x] = x; primes.push(x); mu[x] = -1; } for (const p of primes) { const y = p * x; if (y > N) break; lp[y] = p; if (x % p === 0) { mu[y] = 0; break; } else { mu[y] = -mu[x]; } } } return mu; }

A.2 Exponential Sum Computation

function computeExponentialSum(mu, alpha) { const N = mu.length - 1; let totalReal = 0.0; let totalImag = 0.0; const factor = 2 * Math.PI * alpha; for (let n = 1; n <= N; n++) { if (mu[n] !== 0) { const angle = factor * n; totalReal += mu[n] * Math.cos(angle); totalImag += mu[n] * Math.sin(angle); } } return Math.sqrt(totalReal * totalReal + totalImag * totalImag); }
